# solitadevacademy
Pre-assignment for Solita dev-academy
Assignemnt was to create a web app for displaying Helsinki city-bikes data. Data itself is in CSV-files. 
link to Solita's assignment: https://github.com/solita/dev-academy-2023-exercise

Database working in PhpMyAdmin, docker desktop containers, mysql. 
Database created with migrations and test data uploaded via seeds. 
Testing is made with REST client HTTP-tests.
